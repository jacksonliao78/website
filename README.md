# Personal website

Minimal single-page site built with Vite + React. Pastel blue theme with magenta accents.

## Sections

- About (with photo)
- Projects
- Experience
- Education
- Skills (languages, libraries, frameworks)
- Contact + resume link

## Assets

- **Photo**: Put your photo at `public/photo.jpg` (or update the path in `src/App.jsx`)
- **Resume**: Put your PDF at `public/resume.pdf` so the link works, or set `RESUME_URL` in `src/App.jsx` to an external URL.

## Run

```bash
npm install
npm run dev
```

Build for production: `npm run build`. Output is in `dist/`.
